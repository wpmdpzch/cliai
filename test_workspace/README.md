# cliai 命令测试工作区

此目录为 cliai 命令测试专用，所有操作都在此目录范围内进行，绝不会影响系统和其他文件。

## 目录结构

- `network/` - 网络命令测试
- `file/` - 文件操作命令测试
- `text/` - 文本处理命令测试
- `system/` - 系统管理命令测试
- `encoding/` - 编码命令测试
- `shell/` - Shell 内建命令测试
- `test_results/` - 测试结果输出

## 测试原则

1. 所有文件操作都在 `test_workspace/` 目录下
2. 危险命令（rm, kill 等）只使用绝对安全的目标
3. 网络命令只测试本地或指定的安全目标
4. 测试完成后自动清理测试生成的文件
