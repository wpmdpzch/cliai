## 快捷键定义

| 快捷键 | 功能 |
|--------|------|
| Tab | 切换模式 |
| Ctrl+I | 命令补全 |
| Ctrl+H | 显示帮助 |
| ↑/↓ | 历史记录 |
| Ctrl+C | 退出 |
