# CLI-AI Roadmap

> 让命令行会思考 - Roadmap 规划

---

## 愿景

成为 AI 时代的命令行终极工具：
- 说人话就能操控系统
- 跨平台开箱即用
- 性能与易用性兼得

---

## 版本规划

### v0.1 - MVP (目标: 2026-Q2)

#### Go 版本 - 核心功能
- [ ] 命令行入口 `cliai`
- [ ] **OpenCode 窗口 UI**（Tab 切换 CLI/PLAN/BUILD）
- [ ] 自然语言解析引擎
- [ ] **内置命令包**（curl, jq, grep 等）
- [ ] 原生命令直接执行
- [ ] 基础命令执行
- [ ] OpenAI API 集成
- [ ] 基础配置文件支持
- [ ] 帮助文档

#### Python 版本 - 对齐 Go
- [ ] 基础 CLI
- [ ] 自然语言解析
- [ ] 命令执行
- [ ] API 集成

#### Rust 版本 - 核心框架
- [ ] 项目框架搭建
- [ ] 核心解析逻辑
- [ ] 命令执行

**里程碑**: 能用自然语言执行简单系统命令
